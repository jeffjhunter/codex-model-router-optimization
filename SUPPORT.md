# Support

CMRO is a community project maintained on a best-effort basis.

Use:

- GitHub Discussions for usage questions and routing-design conversations;
- GitHub Issues for reproducible bugs and scoped feature requests;
- private vulnerability reporting for security issues.

Before asking for help, run:

```bash
python routerctl.py doctor --target /path/to/repo
python routerctl.py verify --target /path/to/repo --json
```

Share the CMRO version, operating system, Python version, Codex version, relevant exit code, and redacted output. Never post tokens, credentials, proprietary source, personal absolute paths, or connector data.

The maintainers cannot grant model access, change account entitlements, debug private OpenAI services, or guarantee compatibility with undocumented rollouts. For those issues, use official OpenAI support.
