# Code of conduct

We want this project to be rigorous, welcoming, and useful.

- Discuss ideas and evidence without attacking people.
- Assume good intent while asking for reproducible support.
- Do not harass, threaten, discriminate, or publish private information.
- Respect copyright, licenses, confidentiality, and security disclosure boundaries.
- Accept maintainer decisions about scope, safety, and community health.

Project maintainers may edit or remove contributions and restrict participation when behavior undermines these expectations. Report conduct concerns privately to the repository owner through GitHub. Good-faith reports will be handled as confidentially as practical.
